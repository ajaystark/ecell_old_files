== Installation == 

1. Download [explara-events.zip]
2.Upload the 'explara-events.zip' file from the Plugins->Add New page in the WordPress administration panel.
3.Activate the plugin through the 'Plugins' menu in WordPress.

= Explara Events Work flow Explained =

Once Explara Events plug-in is installed and activated, you can see Explara Events on the sidebar menu section.  

1. Add the token
2. Click fetch & sync to get all the events
