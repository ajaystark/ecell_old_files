== Installation == 

1. Download [explara-lite.zip]
2.Upload the 'explara-lite.zip' file from the Plugins->Add New page in the WordPress administration panel.
3.Activate the plugin through the 'Plugins' menu in WordPress.

= Explara Lite Work flow Explained =

Once Explara Lite plug-in is installed and activated, you can see Explara Lite on the sidebar menu section.  

1. Add the token
2. Add the domain detail
3. Generate shortcode as per requirements.
