== Description ==
Create and Manage your group(s) and grow membership on your website using plugins.


= Salient Features = 

- List all your groups
- Customize the look and feel of the membership page
- Becoming a member of the group and selecting the respective membership plan

= Requirements =

* WordPress 4.6 or greater
* PHP 5.6 or greater (7.0 recommended)
* MySQL 5.2 or greater (5.x recommended)


== Installation == 

1. Download [explara-membership.zip]
2. Upload the 'explara-membership.zip' file from the Plugins->Add New page in the WordPress administration panel.
3. Activate the plugin through the 'Plugins' menu in WordPress.

= Explara Membership Work flow Explained =

Once Explara Membership plug-in is installed and activated, you can see Explara Membership on the sidebar menu section.  

1. Add the token
2. Click fetch & sync to get all the groups
